# iMOSS Core Software

This repository serves as an index of long-term maintained scientific software developed by the iMOSS research group. These tools support research in structural health monitoring, vibration analysis, and physics and data-driven models.

---

## Available Packages

- **pyMLDA** — Machine learning methods for Structural Health Monitoring.
- **Wave** — Wave manipulation and vibration control using smart metastructures.
